# OAuth sample with CellularJS

## I. Get started
1, install PostgreSQL

2, migration
```
yarn migration:run
```

3, start
```
yarn dev
```

## II. Postman
Import `docs/1. api/Cellular OAuth basic.postman_collection.json`
